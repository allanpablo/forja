# Padrões Globais

## Idioma
- Toda comunicação e documentação em **pt-BR**.

## Backend
- API deve ser criada em **NestJS** por padrão.
- Seguir arquitetura modular e testes automatizados.

## Qualidade
- Cada mudança deve registrar impacto, riscos e rollback.
- Evitar soluções ad-hoc; priorizar padrões reutilizáveis.
